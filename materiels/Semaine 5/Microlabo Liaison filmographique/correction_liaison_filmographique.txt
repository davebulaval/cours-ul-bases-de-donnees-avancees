## Instructions pour initialiser l'interface graphique de Neo4J

### 1. Utilisez la commande suivante pour lancer l'application à l'aide de Docker: 
docker run --rm --name neo4j_server -p 7474:7474 -p 7687:7687 -d neo4j

### 2. Sur votre navigateur Web http://localhost:7474

    username: neo4j
    password: neo4j

Pour téléverser les données dans la base de données Neo4j, copier-coller ce qu'il y a dans le fichier donnees.data dans l'invite de commande.

## Questions

1. Ajouter l'acteur "David Sandberg".
CREATE (:Person {name: "David Sandberg"})

2. Ajouter le film "Kung Fury", avec l'année de production 2015.
CREATE (:Movie {title: "Kung Fury"})

3. Trouvez l'acteur "David Sandberg".
MATCH (actor:Person {name: "David Sandberg"}) return actor

4. Trouvez le film "Kung Fury".
MATCH (movie:Movie {title: "Kung Fury"}) return movie

5. Ajouter une relation ACTED_IN allait de "David Sandberg" à "Kung Fury".
MATCH (movie:Movie), (actor:Person ) WHERE movie.title = "Kung Fury" and actor.name = "David Sandberg"
MERGE (actor)-[:ACTED_IN]->(movie)

6. Ajoutez l'étiquette "NO_OSCAR" à "Keanu Reeves".
MATCH (actor:Person {name: "Keanu Reeves"})
SET actor:NO_OSCAR

7. Trouvez les films ayant été réalisés depuis l'an 2000.
MATCH (movie:Movie)
where movie.released > 2000

8. Trouvez tous les films où "Keanu Reeves" a été acteur (relation "ACTED_IN").
MATCH (movie:Movie)<-[:ACTED_IN]-(person:Person  {name: "Keanu Reeves"})
return movie

9. Trouvez le réalisateur (relation "DIRECTED") du film "Top Gun".

MATCH (movie:Movie  {title: "Top Gun"})<-[:DIRECTED]-(person:Person)
return movie, person.name    

10. Combien de critiques (relation "REVIEWED") ont été faites pour le film Jerry Maguire.

MATCH (movie:Movie  {title: "Jerry Maguire"})<-[reviews:REVIEWED]-(person:Person)
return count(reviews)

11. Trouvez tous les films où les acteurs du film "The Matrix Revolutions" ont joué.
MATCH (movie:Movie  {title: "The Matrix Revolutions"})<-[:ACTED_IN]-(person:Person)-[:ACTED_IN]->(movie_played:Movie)
return person, movie_played

12. Ajoutez la paire clé-valeur "seen: True" à tous les films de "Keanu Reeves" (oui vous êtes un fan).
MATCH P = (movie:Movie )<-[:ACTED_IN]-(:Person{name: "Keanu Reeves"} )
foreach(movies in nodes(P) | SET movies.seen )
return person, movie_played
