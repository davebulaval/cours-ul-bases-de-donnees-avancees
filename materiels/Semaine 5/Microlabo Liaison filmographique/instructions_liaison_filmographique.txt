# LABO - Neo4J

## Instructions pour initialiser l'interface graphique de Neo4J

### 1. Utilisez la commande suivante pour lancer l'application à l'aide de Docker: 
docker run --rm --name neo4j_server -p 7474:7474 -p 7687:7687 -d neo4j

### 2. Sur votre navigateur Web http://localhost:7474

    username: neo4j
    password: neo4j

Pour téléverser les données dans la base de données Neo4j, copier-coller ce qu'il y a dans le fichier donnees.data dans l'invite de commande.
